# ANUBIS
Supply Intelligence System - The Arbiter of Routes. The Guardian of Flow.

## Quick Demo (No Setup)
1. Go to the URL (if deployed) or run the code locally.
2. The project is pre-configured with `DEMO_MODE=true` in `.env.example`. When run, it uses robust mock data to demonstrate functionality without needing real API keys.
3. Observe the cinematic splash screen.
4. Interact with the global map, triggering the "Strait of Hormuz" or other chokepoints to visualize cascade effects.
5. Use **The Oracle** to predict risk.

## Installation Local
Run:
```bash
./setup.sh
```
This handles Python, Node packages, and model training.
